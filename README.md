# Architect Finance - Expense Tracker

## Project Overview
**Architect Finance** is a high-fidelity, professional-grade expense tracking web application built as a final project for **MTM6302 - Web Development with JavaScript** at Algonquin College. The application follows an architectural blueprint aesthetic, emphasizing precision, technical hierarchy, and clean typography.

## Features
- **Dynamic Balance Tracking:** Real-time calculation of Total Income, Total Expenses, and Net Balance.
- **Architectural UI:** A custom design system featuring "Blueprint Blue" accents and "Ink Black" surfaces.
- **CRUD Functionality:** Users can add transactions with automatic debit/credit logic and delete entries from the archival log.
- **Responsive Design:** Optimized for both desktop and mobile viewing.

## Technical Implementation
This project was built using **Vanilla JavaScript**, demonstrating mastery in:
- **DOM Manipulation:** Dynamically creating and injecting HTML elements based on state changes.
- **State Management:** Using complex object arrays to handle application data.
- **Math Logic:** Implementing `.reduce()` for financial totals and `.filter()` for categorical data separation.
- **Event Listeners:** Managing user interactions through form submission and dynamic action buttons.

## File Structure
- `index.html`: Main semantic structure of the dashboard.
- `style.css`: Custom "Architect Ink" styling, glass-morphism effects, and animations.
- `app.js`: Core application logic and DOM handling.
- `README.md`: Project documentation.

## Installation & Setup
1. Clone the repository.
2. Open `index.html` in your browser.
3. No build step or installation is required (Vanilla JS).
